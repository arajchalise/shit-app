# 🛠️ Installation

Clone this masterpiece:

```bash
git clone https://github.com/yourusername/shit-app.git
```

Enter the danger zone:

```bash
cd shit-app
```

Install dependencies that may or may not matter:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

If it works first try, something is wrong.
